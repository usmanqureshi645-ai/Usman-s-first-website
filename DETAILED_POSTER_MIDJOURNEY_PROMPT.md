## HOW TO EXECUTE

### Using Midjourney:
1. Copy the "MIDJOURNEY v6 PROMPT" above
2. Go to Discord → Midjourney Bot → `/imagine`
3. Paste the prompt
4. Wait 60 seconds for generation
5. Upscale your preferred option (U1, U2, U3, or U4)
6. Right-click the final image → "Open in New Tab"
7. Save as PNG (high resolution)
8. **Cost:** ~$0.40 per image

### Using Flux:
1. Copy the "FLUX PRO PROMPT" above
2. Go to [Flux Pro](https://flux-pro.ai)
3. Paste the prompt
4. Generate (10-30 seconds)
5. Download high-res PNG
6. **Cost:** $1-2 per image

### Using Leonardo.ai:
1. Copy the "LEONARDO.AI PROMPT" above
2. Go to [Leonardo.ai](https://leonardo.ai)
3. Create new canvas
4. Paste prompt into text field
5. Select "Leonardo Vision" model
6. Generate (2-3 minutes)
7. Download high-res PNG
8. **Cost:** Free tier / $10-20/month

## LINKEDIN POSTING SPECS

**Image Size:** 1200×2640px (vertical, 1:2.2 ratio)  
**Format:** PNG (recommended) or JPG  
**File Size:** Keep under 5MB (PNG auto-compresses in LinkedIn)

**Why Vertical?**
- 46% more impressions than horizontal
- Optimized for mobile (80% of LinkedIn traffic)
- Fills more of the feed, stops the scroll
